Cursos

Aqui estan algunos de los cursos que tenemos

Front-End

Vas a aprender a programar en HTML5, CSS3 y JavaScript en un mes y medio, terminando con la habilidad para desarrollar el Front End de sitios complejos y aplicaciones web. Este conocimiento te permite dar tus primeros pasos como programador, o postularte al Coding Bootcamp para dar un salto de carrera.

Back-End

Vas a aprender a utilizar JavaScript para programar del lado del servidor, utilizando NodeJS. Vas a poder construir un servidor web con express. Durante el curso vas a desarrollar un proyecto integrador que incluye templating engines para las vistas, y MongoDB para la persistencia de datos.

Word-Press
